# Research Notebook Template

A starter template for academic research and literature reviews.

## Features

- Warm, scholarly "manuscript" theme
- Organized for paper summaries and research notes
- Claude Code integration for AI-assisted research

## Getting Started

1. Copy this folder to your project: `cp -r examples/research-notebook my-research`
2. Open `index.html` in Chrome/Edge
3. Select your copied folder
4. Start adding notes and papers

## Suggested Workflow

1. Create a bookmark when you find an interesting paper
2. Add a note summarizing key findings
3. Use code cells for data analysis
4. Link related notes with `[[Section > Title]]`
