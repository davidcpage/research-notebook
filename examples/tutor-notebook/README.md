# Tutor Notebook Template

A starter template for AI-assisted learning and tutoring sessions.

## Features

- Friendly, accessible color theme
- Organized for learning topics and exercises
- Claude acts as a patient tutor

## Getting Started

1. Copy this folder to your learning project: `cp -r examples/tutor-notebook my-learning`
2. Open `index.html` in Chrome/Edge
3. Select your copied folder
4. Start learning!

## Suggested Workflow

1. Create notes for topics you want to learn
2. Ask Claude to explain concepts step by step
3. Use code cells to practice and experiment
4. Track your progress with tags: `beginner`, `intermediate`, `completed`
