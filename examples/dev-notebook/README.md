# Development Notebook Template

A starter template for coding projects and technical documentation.

## Features

- Dark terminal-inspired theme
- Organized for code and technical docs
- Optimized for Claude Code integration

## Getting Started

1. Copy this folder to your project: `cp -r examples/dev-notebook my-project-notes`
2. Open `index.html` in Chrome/Edge
3. Select your copied folder
4. Start documenting

## Suggested Workflow

1. Create notes for architecture decisions
2. Add code cells for examples and prototypes
3. Bookmark API docs and references
4. Use tags to track status: `ongoing`, `completed`, `future`
